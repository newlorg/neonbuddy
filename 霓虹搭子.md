# PRD v1.0 - 《霓虹搭子：RushLab》

## 1. 文档目标
提供一款面向年轻人（不分男女）的新爆款方向，重点验证“快节奏 + 强社交 + 高表达 + 可长期运营”。

## 2. 产品定位
- 类型：3D轻竞技动作 + 组队夺点 + 潮流表达
- 核心体验：3-5分钟一局，操作简单但上限高，和朋友开黑有明显乐趣
- 平台：iOS/Android
- 商业模式：IAP为主，激励广告为辅

## 3. 目标用户与场景
### 3.1 用户画像
- 16-30岁为核心
- 喜欢短视频平台风格的快反馈内容
- 既要“能秀操作”，也要“能秀审美和人设”

### 3.2 核心场景
- 通勤和课间：3分钟快速开一局
- 晚间社交：双排/四排开黑
- 周末活动：参加赛季活动和社区挑战图

## 4. 核心设计原则
1. 上手30秒：移动、冲刺、技能释放全是低门槛手势
2. 失败不坐牢：落后方有追击机制，减少一边倒
3. 社交先于数值：组队收益明显高于单人肝
4. 个性表达可沉淀：外观、动作、称号、主页空间可长期积累
5. 公平优先：不卖破坏对局公平的数值

## 5. 玩法方案
### 5.1 核心模式：`能量争夺战`（4v4，单局4分钟）
1. 地图中立刷新能量芯片
2. 队伍需运送芯片回己方核心站点
3. 可通过技能干扰、拦截、偷运完成反打
4. 时间结束或先达目标分的一方获胜

### 5.2 单局节奏
1. 0:00-0:30 抢开局资源
2. 0:30-2:30 中盘拉扯与小团战
3. 2:30-4:00 终局高价值芯片刷新，形成翻盘点

### 5.3 角色与技能
- 初始8名角色，定位分为突进/控场/支援/干扰
- 每角色3个主动技能 + 1个被动
- 技能构成强调“看得懂的效果”，避免学习成本过高

## 6. 年轻化差异点
1. `搭子系统`：基于开黑记录自动推荐高默契队友，降低组队成本
2. `高光回放卡`：系统自动截取精彩10秒，支持一键生成分享短视频
3. `潮流工坊`：玩家可组合贴纸、喷漆、动作，生成个人风格套装
4. `社区挑战图`：官方每周精选玩家地图玩法，增强内容新鲜度

## 7. 社交与社区
### 7.1 轻社交
- 一键双排和附近战绩匹配
- 赛后“继续组队”强引导
- 搭子亲密度等级与专属奖励

### 7.2 中度社群
- 30人“街区团”，周任务合作完成
- 团队排行榜偏协作指标，不只看击杀
- 团队语音与文本双通道，默认开启文明聊天过滤

## 8. 成长与留存
- 日常成长：角色熟练度、通行证等级、任务链
- 中期目标：段位、搭子默契等级、工坊收藏完成度
- 长线目标：赛季称号、限定风格套装、个人主页陈列

## 9. 经济与变现
### 9.1 货币体系
- 信用点：日常成长
- 潮流券：外观与动作兑换
- 晶钻：高级外观和通行证

### 9.2 付费结构
- 赛季通行证（主收入）
- 限定外观包（联动/节日）
- 主页空间装饰包（高表达付费点）
- 搭子双人礼包（偏社交场景）

### 9.3 广告策略
- 仅激励视频：赛后额外奖励、任务加速、回流补给
- 竞技主流程不插广告

## 10. LiveOps规划
- 每周：新挑战图 + 限时玩法轮换
- 每月：主题版本（新角色或新地图）
- 每赛季（8周）：段位重置 + 赛季剧情 + 大型活动

## 11. 首发90天KPI
- D1 >= 48%
- D7 >= 26%
- D30 >= 11%
- 开黑渗透率 >= 45%
- 付费转化 >= 3.5%
- 通行证购买率 >= 10%
- 赛季活动参与率 >= 65%

## 12. 风险与对策
- 风险：轻竞技环境可能被高玩垄断
- 对策：隐藏分匹配 + 新手保护池 + 追击机制
- 风险：内容更新压力高
- 对策：官方地图 + 社区精选双轨供给
- 风险：社交内容失控
- 对策：强审核词库、举报闭环、分级处罚

## 13. 里程碑建议
1. P0（2周）：核心移动与夺点玩法原型
2. P1（4周）：4v4对局闭环 + 角色技能框架
3. P2（6周）：搭子系统 + 通行证 + 基础商城
4. P3（8周）：社区挑战图 + 首轮灰度测试
5. P4（10-12周）：商业化调优 + 区域上线准备
